# Task Management
 O task management é um projeto para melhorar organização de tarefas no dia a dia.
 Você pode se cadastrar no TaskManagement e organizar suas tarefas.

### Tecnologias
  - Typescript
  - Node
  - NestJs

 
### Funcionaliades
  - Cadastrar Usuario
  - Login
  - Controle de Tarefas (Cadastrar, Deletar, Atualizar, Listar)
